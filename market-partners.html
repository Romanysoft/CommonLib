A.【工作指标结算报酬】

工作指标: 由我们分配的Google Blogger博客站点（只含有我们提供的产品推广内容，且主要推广到欧美等国家），以邮件中帖子描述内容为准。通过Google官方分析工具（分析信息会共享给你）对浏览用户数的统计计算办法，每月报酬计算方法如下：

  • 1)Blogger站点每个浏览用户，报酬为 0.025 RMB。
  • 2)Blogger站点每月推广最高得到的劳动报酬为 2000 RMB。

说明：一般情况下，一个FaceBook小组人数都超越了2000人，最多人数达到上百万人。如果，你拥有50个小组以上，可以想象一下，人数有多么众多。分享帖子信息在FaceBook小组中，就是希望该小组的人能够看到，并浏览这个帖子。如果有一个人浏览了，那么就是一个浏览用户；如果有上万个人浏览，就是上万个浏览用户。自己可以计算一下，如果一天要拿到报酬为50元，是不是只需要让2000人浏览帖子就可以了。我们衷心希望，你分享的帖子有更多人看到。这样，也会更容易为你带来订单，从而实现你和我们的利益双赢。

B.【销售订单额提成结算报酬】

销售结算以销售跟踪ID号在订单查询系统中查询到的数据为准（以自然月为统计单位）

  • 汇率参考网址：https://www.oanda.com/lang/cns/currency/converter/
  • 美元结算人民币以结算时的汇率为准。

订单销售额 = 用户购买订单总额 - 用户退单额总额。

  • 订单销售额 < 5千，报酬=订单销售额*20%；
  • 5千 <= 订单销售额 <1万，报酬=订单销售额*25%；
  • 1万 <= 订单销售额 <3万，报酬=订单销售额*35%；
  • 3万 <= 订单销售额 <7万，报酬=订单销售额*45%；
  • 订单销售额 >= 7万，报酬=订单销售额*50%；

C.【招募人员提成结算报酬】

  • 报酬 = 你所招募人员月劳动报酬总计*15%；
    例如：你招募了两个人，（1）王军；（2）李梅。 7月份，王军拿到的月最终劳动报酬为4000RMB，李梅拿到的月最终劳动报酬为5000RMB。那么，你，7月份，能拿到的招募人员月劳动报酬为： (4000 + 5000) * 15% = 1350 RMB.

月最终报酬 = A.工作指标结算报酬 + B.销售订单额提成结算报酬 + C.招募人员提成结算报酬。

举例：7月份，你订单销售拿到的报酬为3500 RMB，你工作指标结算报酬为2000 RMB，你招募人员劳动报酬提成为2500RMB。那么你7月份劳动报酬总计：3500 + 2000+ 2500= 8000 RMB。
